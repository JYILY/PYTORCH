# PYTORCH
MY PROJECT WITH PYTORCH

带图形界面的手写数字识别小程序 <br/>
花了一天的时间写着玩玩 <br/>

运行方法 ：<br/>
直接运行main.py就好了（前提是你得有各种包，例如torch）<br/>

该程序里面包含两个模型 ：CNN 和Linear <br>

* CNN的训练通过率和测试通过率都超过 99%  <br>
* Linear的训练通过率和测试通过率都超过 98% <br>
* 但好像识别自己写的有时候有超级玄学的错误，究其原因，应该是mnist的数据集是来自于外国人的手写数字习惯吧 <br>
